## Prerequisites:
  [Yarn](https://classic.yarnpkg.com/en/docs/install#debian-stable)
  [nodeJs Lts(14.15.X)](https://nodejs.org/en/download/)

## How to use:

#### Clone repository
```git clone https://github.com/nik-gol/young-app-test.git```

#### Install project dependecies
```yarn install```

#### Run app
```node src/api.js```

#### Run tests
```yarn test```
